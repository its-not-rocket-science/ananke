# API Reference (Generated)

This directory is generated from JSDoc comments via TypeDoc during release builds.

## Generate locally

```bash
npm run docs:api
```

## Try-it links

Each generated function page appends a link in this format:

```txt
https://codesandbox.io/s/github/its-not-rocket-science/ananke/tree/main/examples/reference/tactical-duel
```

Release automation writes generated files to `docs/api/` and publishes with the docs site.
