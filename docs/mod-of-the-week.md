# Mod of the Week

Every week we highlight one community-made content pack that is fun, creative, or technically interesting.

## How to submit

1. Create a content pack JSON (archetypes, weapons, armour, terrain, or all four).
2. Add a short description:
   - Theme
   - What makes it interesting
   - Recommended seed(s)
3. Open a PR with:
   - File under `examples/custom-content/`
   - A replay clip or JSON export
   - A short note on expected balance/fun factor

## Selection criteria

- Works with current schema validation.
- Replays deterministically with provided seed.
- Adds clear creative value for tinkerers.

## Badge usage

The README badge links to this page and invites submissions:

- Label: `Mod of the Week`
- CTA: `Submit Yours`

Selected mods will be listed here as we begin the weekly showcase cycle.
