Ananke documentation is now organized by **role-based learning paths**, with hands-on checkpoints and expected outputs.

## Start by role

- **Game developers**: build a deterministic multiplayer tactics game in two hours.
- **ML researchers**: train an agent in a Gymnasium-compatible combat environment.
- **Hobbyists/modders**: build and publish your first content pack.
- **Storytellers**: generate narrative combat logs and export to Obsidian.
